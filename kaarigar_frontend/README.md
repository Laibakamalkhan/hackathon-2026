# KARIGAR — Flutter Mobile Frontend

Production-ready Flutter app (Android + iOS) for **KARIGAR**, rebuilt from the Figma reference in `../figma_design_code/`.

## Run

```bash
cd ai_seekho_flutter_frontend
flutter pub get
flutter run
```

## Onboarding flow

1. Splash → **Language** (first)
2. Tutorial (with Skip)
3. Role selection (seeker / provider)
4. Phone auth → OTP
5. Profile setup (name, city, area, street address)
6. Home or Provider dashboard

## Stack

- Flutter / Dart
- Riverpod
- go_router
- google_fonts (Nunito)
- flutter_animate
